Kotlin

C’est un langage de programmation orienté objet et fonctionnel avec un typage statique et permet de compiler pour la JVM

Il vient combler les défauts de Java pour la programmation sur Android, qui est un langage lourd et verbeux. Kotlin adopte donc une syntaxe plus simple. JetBrains a pensé à la compatibilité entre les deux langages, ce qui permet d’utiliser les deux dans un même projet.

    Avantage

  • Syntaxe peu verbeuse (déclarations plus courtes, pas d’accesseurs, etc)
  • Inférence de types
  • Interopérable avec java
  • Pas de Null
  • gratuit et open source

    Inconvénient

  • Peut générer du code difficile à relire
  • Des comportements par défaut gênants. (exemple: les classes sont finales par défaut)
  • Jeune langage, il y a donc encore assez peu de documentation

Comparaison Kotlin/Java

Kotlin:


 fun main() {
  println("Hello, World!")
 }

Java

 class HelloWorld {
  public static void main(String[] args) {
   System.out.println("Hello, World!");
  }
 }

Les variables

Kotlin connaît deux types de variables :
  • Les variables qui sont en lecture seul: val (pour valeur)
  • Les variables qui sont modifiable: var (pour variable)

    • val name: String = "Phil"
    var age: Int = 27

    Les Conditions et boucles

    Condition if

     var max: Int

     if (a > b) {
      max = a
     } else {
       max = b
     }

    Boucle For

     for (i in array.indices) {
       println(array[i])
     }

    Boucle While

     while (x > 0) {
       x--
     }
